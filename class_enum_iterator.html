Detailed Description

template<class T, T start, T stop>
class EnumIterator< T, start, stop >

Iterator for an enum of type T that starts with an item start and ends after the item stop. Models ForwardIterator concept.

Warning
Following iterator DOES NOT WORK for non-contiguous enumerations like "enum class E { A = 100, B = 200, C == 300 }".
See also
http://en.cppreference.com/w/cpp/concept/ForwardIterator
